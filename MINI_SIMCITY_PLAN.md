# CivilSim Mini SimCity 기획서 (확정안 v1)

이 문서는 개발 착수 전 범위를 고정하기 위한 기준 문서다.

## 1. 제품 방향

- 장르: 미니 시티빌더 (심시티 감성, 경량 시뮬레이션)
- 핵심 재미:
  1. 플레이어가 도로/지반/건물을 직접 배치해서 도시를 키운다.
  2. 숫자(예산, 인구, 만족도)가 배치 결과로 즉시 피드백된다.
  3. 작은 선택이 월 단위 운영 결과로 연결된다.

## 2. 절대 원칙 (이번 버전)

- 자동 개발 금지:
  - 구역(R/C/I)을 지정해도 건물이 자동으로 생성되지 않는다.
  - 모든 건물은 플레이어가 수동 배치한다.
- 복잡한 인프라 네트워크(전선/파이프 시각망)는 v1 범위 제외.
- 차량 에이전트 교통 시뮬레이션은 v1 범위 제외(수치 기반 혼잡도만 고려 가능).

## 3. MVP 플레이 루프

1. 지반 정비(G)
2. 도로 배치(F)
3. 구역 지정(Z, R/C/I/X)
4. 건물 수동 배치(B)
5. 시간 진행(월 정산)으로 도시 상태 확인
6. 철거/재배치로 최적화

## 4. 기능 범위

## 4.1 포함

- 그리드/카메라/시간 배속/HUD
- 지반 정비(직사각형)
- 도로 배치/철거(직선 드래그)
- 구역 지정/해제(직사각형)
- 건물 수동 배치/철거
- 월별 경제 정산(수입/지출/잔액)
- 기본 알림(자금 부족/배치 불가/모드 전환)

## 4.2 제외

- 구역 자동 개발
- 전기/수도 실제 연결망
- 시민 개별 에이전트 이동
- 재해 시스템
- 저장/불러오기(후순위)

## 5. 시스템 규칙 (v1)

## 5.1 구역 규칙

- ZoneType: Residential / Commercial / Industrial / None
- Road/Building 셀에는 구역 지정 불가
- Foundation/Empty/Zone 셀에만 지정 가능
- 구역은 정보 레이어이며, 자동 건설 트리거로 사용하지 않는다.

## 5.2 건물 배치 규칙

- 건설은 Foundation 셀에서만 가능
- 도로 인접 여부를 건물 배치 조건으로 둘지 여부는 옵션:
  - 기본: 배치는 허용, 운영 효율에 패널티를 부여
- 자금 부족 시 배치 실패

## 5.3 경제 규칙

- 수입: (주거 인원 * 주거세) + (고용 인원 * 사업세)
- 지출: 건물 월 유지비 합
- 파산 임계치 이하 시 경고 상태
- 게임오버는 보류 (당장 종료시키지 않음)

## 5.4 인구/수요(경량)

- 인구는 수동 배치 건물의 ResidentCapacity 합으로 시작
- 수요 지표:
  - 주거 수요 = 고용 - 거주
  - 상업/공업 수요 = 거주 기반 파생값(단순식)
- 수요는 UI 수치로만 먼저 노출

## 6. UI/입력 스펙

- B: 건물 패널 토글
- F: 도로 모드 토글
- G: 지반 모드 토글
- Z: 구역 모드 토글
- R/C/I/X: 구역 타입 전환/해제 (Z 모드에서)
- ESC/RMB: 현재 모드 취소

## 7. 콘텐츠 스코프 (현재 에셋 기준)

- 건물 카테고리: 주거/상업/공업만 우선
- 공공/유틸리티는 데이터 슬롯만 예약하고 구현 보류
- 사운드 에셋 부재: v1은 무음 또는 임시 클릭 SFX(후속)
- 아이콘 부족: 텍스트+색상 기반 UI 우선

## 8. 개발 순서 (실행 백로그)

## Sprint 1: 루프 안정화

1. Zone 모드 입력/미리보기/적용 안정화
2. 모드 충돌 제거(B/F/G/Z 동시 활성 방지)
3. HUD에 현재 모드/구역 타입 표시

## Sprint 2: 운영 지표

1. 인구/고용/수요 계산기 추가
2. 월 정산에 수요 반영 계수 추가
3. 수치 알림 정리(적자 지속, 수요 과열)

## Sprint 3: 품질

1. 튜토리얼 툴팁(단축키 안내)
2. 밸런스 테이블 1차 조정
3. 플레이테스트 체크리스트 작성

## 9. 완료 기준 (Definition of Done)

- 15분 플레이 세션 동안:
  - 플레이어가 지반-도로-구역-건물을 반복해 도시를 확장할 수 있다.
  - 월 정산 수치가 변하고, 플레이어가 재배치로 개선할 수 있다.
  - 치명적인 소프트락/모드 충돌이 없다.

## 10. 보류 이슈 (다음 의사결정)

- 도로 미연결 건물을 "배치 불가"로 할지 "운영 패널티"로 할지
- 파산 시 즉시 실패 처리 여부
- 공공/유틸리티를 v1.1에서 최소 구현할지 여부
