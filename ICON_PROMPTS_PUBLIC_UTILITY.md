# Public and Utility Building Icon Prompts

## Global style prompt
Use this prefix for all icons:

`Clean city-builder UI icon, isometric 3/4 view, soft toon shading, simple readable silhouette, minimal details, centered composition, transparent background, no text, no logo, no border, consistent lighting from top-left, game asset icon`

## Negative prompt
Use this suffix for all icons:

`photorealistic, cluttered background, watermark, words, letters, logo, frame, low resolution, blurry, noisy, oversaturated, realistic people, dramatic perspective`

## Output spec
- Size: 1024x1024
- Background: transparent PNG
- Object scale: 70 to 80 percent of frame
- Color profile: sRGB

## Per building prompts

### 1) Elementary School (Public)
`A small modern elementary school building with a clean facade, bright windows, tiny playground hint, welcoming civic design, city simulation icon`

### 2) Community Clinic (Public)
`A compact neighborhood medical clinic, white and light-blue exterior, clear entrance canopy, emergency cross symbol integrated on signage shape without text`

### 3) Fire Station (Public)
`A low-rise urban fire station with red accents, wide garage door for fire trucks, emergency response building silhouette, civic service style`

### 4) Power Substation (Utility)
`A compact electrical substation, fenced transformer yard, utility boxes, warning stripe accents, industrial civic infrastructure icon`

### 5) Water Pump Station (Utility)
`A municipal water pump station with tanks and pipes, clean industrial architecture, blue utility accents, infrastructure building icon`

### 6) Waste Processing Center (Utility)
`A small waste processing facility with sorting containers and ventilation units, neutral industrial colors, clean stylized city-builder icon`
